Stuff I know

AngularJS

An open source framework developed to help web developers separate their models, views and controllers when building single page applications. I've been using it in most of my projects as of late and it's become an invaluable tool!

ReactJS

React is, like Angular, a framework to assist when building single page applications. It's a relatively new tool in my toolbox, but one I'm getting more and more fond of.

Bootstrap

Another open source front-end framework, bootstrap assists the developer in creating consistent and responsive interfaces. Another tool that's become an essential part of my everyday work.

C#

In my days I have worked with several programming languages, but C# is the one I've grown to like the most. Microsoft have done an excellent job of providing us developers with the tools and environments we need to be as productive as possible while delivering high quality products!

ASP.NET MVC

ASP.NET MVC is a web application framework which implements the Model-View-Controller pattern. Combined with the tools and utilities of Visual Studio I am able to build advanced back-end applications in no time!

ASP.NET WebApi

Much like MVC, the WebApi provides structure and tools to grant front-end applications access to resources and functions on the webserver or database.
